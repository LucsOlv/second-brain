# 🗄️ Arquivo

Notas antigas, projetos concluídos e material inativo.

## Regra

Arquivar nota quando:
- Projeto finalizado há mais de 3 meses
- Tarefa cancelada ou duplicada
- Referência já incorporada em nota própria

## Como arquivar

1. Mover o arquivo para `99-Arquivo/`
2. Remover de MOCs relevantes
3. Adicionar tag `#arquivado` no frontmatter
4. Manter os links (não quebrar conexões)

## NÃO arquivar

- Notas de projetos ainda ativos
- Referências que ainda são consultadas
- Check-ins de saúde mental (manter histórico)
