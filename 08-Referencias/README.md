# 📚 Referências

Material de estudo, artigos, vídeos, livros — tudo para referência futura.

## Template

```yaml
type: reference
title: 
source: # article/book/video/podcast/course
url: 
author: 
date: YYYY-MM-DD
tags: []
summary: |
  Resumo em 2-3 linhas.
key_takeaways:
- 
related_notes:
- 
```

## Tipos de source

- `article` — artigo online ou PDF
- `book` — livro lido ou para ler
- `video` — YouTube, aula, talk
- `podcast` — episódio de podcast
- `course` — curso online

## Regras

- Linkar notas que referencing this material
- Usar tags para filtrar por tema
- Manter `key_takeaways` curtos (bullet points)
