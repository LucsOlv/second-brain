# 📅 Daily Notes

Notas do dia a dia. Cada arquivo corresponde a uma data.

## Template

```yaml
type: daily
date: YYYY-MM-DD
energy: # 1-10
mood: # happy/neutral/low/anxious
focus_today: 
accomplished:
- 
next_actions:
- 
```

## Conventions

- Nome do arquivo: `YYYY-MM-DD.md`
- Linkar projetos ativos do dia
- Linkar pessoas encontradas
- Registrar energia e humor
- Fazer review do que foi feito
