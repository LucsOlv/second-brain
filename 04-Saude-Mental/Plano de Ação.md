# 🧭 Meu Plano de Ação

> **Passos concretos a seguir quando sentir que estou a precisar de ajuda.**
> Parte do [[Plano de Segurança]]

---

## Nível 1 — Sinais Leves

> *Estou a sentir que algo não está bem, mas consigo gerir.*

**Passos:**
1. [ ] Fazer pausa — respirar fundo 5x
2. [ ] Identificar o que estou a sentir (nomear reduz a intensidade)
3. [ ] Fazer uma atividade breve de distração ([[Formas de Me Manter Ocupado]])
4. [ ] Se não passar em 30min, avança para Nível 2

---

## Nível 2 — Sinais Moderados

> *Os sinais estão a piorar ou persistem.*

**Passos:**
1. [ ] Primeiro tentar: técnica de grounding (5-4-3-2-1)
2. [ ] Ligar para [[Rede de Apoio]] — contactar a primeira pessoa disponível
3. [ ] Usar uma [[Habilidades de Enfrentamento|técnica de enfrentamento]]
4. [ ] Evitar decisões importantes até estar mais calmo
5. [ ] Se não melhorar em 2h, avança para Nível 3

---

## Nível 3 — Crise Aguda

> *Estou a perder o controlo ou tenho pensamentos de dano.*

**Passos:**
1. [ ] **Ligar para linha de crise: 112** ou [[Recursos Úteis|Linhas de Apoio]]
2. [ ] Contactar [[Rede de Apoio|psicólogo (Leonardo)]] ou familiar
3. [ ] Ir para lugar seguro — longe de objetos perigosos
4. [ ] Não ficar sozinho — manter alguém por perto
5. [ ] Lembrar: **isto vai passar. Não precisas de resolver tudo agora.**
