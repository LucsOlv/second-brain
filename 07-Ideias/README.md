# 💡 Ideias

Banco de ideias — tudo que vier à cabeça. Não filtrar aqui.

## Template

```yaml
type: ideia
title: 
status: # raw/developed/rejected/implemented
tags: []
created: YYYY-MM-DD
description: |
  Describe the idea here.
why_good: |
  Why is this interesting?
```

## Stages

1. `raw` — só a semente da ideia
2. `developed` — refinada e com contexto
3. `rejected` — não faz sentido
4. `implemented` — foi para produção/algo real

## Regras

- Não julgar aqui. Tudo é válido.
- Revisar mensalmente para matar ideias mortas
- Linkar ideias relacionadas entre si
