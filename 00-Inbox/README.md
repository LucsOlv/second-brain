# 📥 Inbox

Pasta de captura rápida. Tudo que entra aqui deve ser processado e movido para o local certo.

## Regra do Inbox Zero

1. Toda ideia/lembrete vai primeiro aqui
2. Processar diariamente
3. Mover para a pasta correta
4. Arquivar ou linkar

## Tags comuns

- `#captura` — ideia pendente de processar
- `#urgente` — precisa resolver hoje
- `#delegado` — passado para outra pessoa
