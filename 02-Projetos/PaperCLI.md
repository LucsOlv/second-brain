## 🎯 Objetivo

Facilitar pesquisa académica: buscar, organizar e analisar papers diretamente do terminal.

## 📋 Uso

```bash
# Ver ajuda
paper-cli --help

# Buscar papers
paper-cli search <query>

# Listar biblioteca
paper-cli list
```
